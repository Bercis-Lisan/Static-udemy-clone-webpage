# Static-udemy-clone-webpage

Udemy Clone (Static Website)

A responsive static clone of the Udemy landing page built using HTML and CSS. This project replicates the design and layout of Udemy's homepage, including the navigation bar, course sections, categories, promotional banners, and footer. The focus was on practicing front-end development skills, responsive design principles, and creating a clean user interface without any backend functionality.

Features
Responsive layout for different screen sizes
Modern navigation bar
Course cards and category sections
Hero/banner section
Clean and user-friendly UI
Built with pure HTML and CSS
Technologies Used
HTML5
CSS3
Purpose

This project was created for learning and practicing front-end web development by recreating the look and feel of a popular online learning platform.

Note: This is a static educational project and is not affiliated with or endorsed by Udemy.
